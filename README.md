# 個別ざぶとんv6  
指定した文字にざぶとんを複数表示できるスクリプト。  
こちらは複数のざぶとんが表示でき、ざぶとんの位置と大きさは自動できまる。  
こちらはAIの助力を大きく借りた。ほとんどAIに作ってもらった。  
その時に使用したAIはgoogleの「Antigravity」である。  
最初にアップロードした時点からいきなりv6であるが、それの前にローカル環境に試作品が5つあるからである。  
試作品の方は上手く動かなかったりするためアップロードの予定はないが、そのうちv5だけは(ひな型)として、サンプル用に同時にアプロードしておく。  

# 個別ざぶとんv5(ひな型)   
「個別ざぶとんv6」を作るとき、「Antigravity」に見せたひな型がこちらとなる。  
うまく機能しないため、使用する目的ではなく、ここから改良してもらったというサンプルとなる。  

二次配布可、改変可、クレジット不要
